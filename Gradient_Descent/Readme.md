# Gradient Descent From Scratch

## Overview

This project implements Gradient Descent for Linear Regression completely from scratch using Python and NumPy. It also compares the scratch implementation with scikit-learn's Linear Regression model.

## Features

- Dataset loading
- Data overview
- Feature selection
- Train-test split
- Gradient Descent implementation
- Model evaluation
- Scratch vs scikit-learn comparison
- Visualization of training results
