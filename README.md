# 🧠 Machine Learning From Scratch

> A comprehensive collection of Machine Learning algorithms implemented from scratch using **Python** and **NumPy**, along with comparisons against **Scikit-learn** implementations to strengthen mathematical intuition and algorithmic understanding.

---

## 📌 About This Repository

Machine Learning libraries make it easy to build models in just a few lines of code. However, understanding **how these algorithms actually work internally** is what builds strong machine learning fundamentals.

This repository focuses on implementing popular Machine Learning algorithms **from scratch**, without relying on Scikit-learn's built-in algorithms for training.

Each implementation is later validated by comparing it with the corresponding Scikit-learn implementation.

---

## 🎯 Repository Objectives

- Understand Machine Learning algorithms mathematically.
- Implement algorithms completely from scratch.
- Compare custom implementations with Scikit-learn.
- Build strong problem-solving and implementation skills.
- Develop a solid foundation before moving towards Deep Learning.

# 🚀 Why This Repository?

This repository is designed for students and aspiring Machine Learning Engineers who want to understand the mathematical intuition behind Machine Learning algorithms rather than simply using libraries.

Every algorithm is implemented from scratch before comparing its performance with Scikit-learn.
